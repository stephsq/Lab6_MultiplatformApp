## About The Project

The Recipe Explorer is a multiplatform application built using **Kotlin Multiplatform** and **Compose Multiplatform**. It allows users to browse a collection of recipes and view their details. 

## How to Run

### Android
1. Open the project in Android Studio.
2. Select the **`composeApp`** run configuration with the Android icon.
3. Select an Android Emulator from the device dropdown.
4. Click **Run**

### Desktop (JVM)
1. Select the **`composeApp: jvm`** run configuration with the Android icon.
2. Click **Run**

### Web (wasmJs)
1. Select the **`composeApp: wasmJs`** run configuration with the Android icon.
2. Click **Run**
